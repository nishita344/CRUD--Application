# CRUD Application

A full-stack Users Manager built with Node.js, Express, LocalStorage and SessionStorage.

## 🚀 Live Demo
👉 [https://crud-application-scjh.onrender.com/](https://crud-application-scjh.onrender.com/)

## Tech Stack
- Backend: Node.js, Express.js
- Storage: LocalStorage (persistent) + SessionStorage (draft)
- Frontend: HTML, CSS, Vanilla JavaScript

## Features
- Add, View, Update, Delete users
- No database needed — data stored in browser
- Auto-save form draft with SessionStorage
- Dark pro UI with search and stats
- Fully responsive

## Run Locally
```bash
npm install
node server.js
```
Open `http://localhost:3000`
